Ram

RAM acts like short-term memory for your computer. This memory offers performance boosts due to the extremely fast data access. RAM is very important for video editing, photo editing, gaming, and rendering. The more RAM you have, the more programs and browser tabs you can have at the same time. It is better to have two sticks of 8gb RAM than one stick of 16gb RAM; the reason for this is because of multi-channel memory. Multi-channel memory has every stick of RAM have a channel of communication with the CPU.

DDR4 vs DDR5 RAM

DDR4 was released in 2014, while DDR5 was released in 2020. DDR4 is still used commonly in computers due to it being compatible with most motherboards and the RAM being cheaper. DDR5 is still the better option if your computer supports it and you have the extra cash.
